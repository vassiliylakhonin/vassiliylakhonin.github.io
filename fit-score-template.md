# Fit Score Template (0.0–5.0)

Use this before any application.

## Weighted dimensions
- Role match (scope/seniority): **25%**
- Core capability match (delivery/ops/compliance): **20%**
- Industry transferability: **10%**
- Region/timezone/work mode fit: **10%**
- Compensation/title fit: **10%**
- Risk constraints (visa/travel/language/contract): **10%**
- Growth and strategic value: **15%**

## Scoring rubric
- **4.5–5.0**: Priority A (apply now)
- **4.0–4.4**: Priority B (apply with tailored CV)
- **3.5–3.9**: Conditional (apply only with strong narrative bridge)
- **<3.5**: Skip
