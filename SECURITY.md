# Security Policy

## Supported scope

Security issues are relevant for:
- `mcp/server.py` and transport settings
- exposed machine-readable endpoints
- CI/CD workflow integrity
- dependency vulnerabilities affecting runtime behavior

## Reporting a vulnerability

Please do not open a public issue for security problems.

Report privately to:
- Email: `vassiliy.lakhonin@gmail.com`
- Subject: `[SECURITY] <short summary>`

Include:
- affected file/path
- impact
- reproduction steps
- suggested fix (if available)

## Response targets

- Initial acknowledgement: within 72 hours
- Triage decision: within 7 days
- Fix timeline: depends on severity and complexity

## Disclosure

After a fix is available, coordinated disclosure is welcome.
