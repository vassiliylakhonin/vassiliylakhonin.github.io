# Code of Conduct

## Our commitment

We want this project to be useful, respectful, and inclusive for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus feedback on ideas and outcomes.
- Assume good intent, ask clarifying questions first.
- Share evidence for technical claims.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Personal attacks, trolling, or repeated bad-faith arguments.
- Doxxing or sharing private information.
- Spam or intentionally disruptive behavior.

## Enforcement

Project maintainers may remove comments, close threads, or block contributors for violations.

## Reporting

Report serious issues privately via security contact in `SECURITY.md`.
