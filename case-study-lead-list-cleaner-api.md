# Case Study: Lead List Cleaner API

## What it solves
Lead imports frequently contain invalid, duplicate, or low-quality records that waste campaigns.

## What I built
A production-ready API (`/v1/clean`) with dedupe, validation, and keep/drop/review decisioning + scoring for each record.

## Why it matters
Improves list hygiene and reduces wasted outbound spend.
